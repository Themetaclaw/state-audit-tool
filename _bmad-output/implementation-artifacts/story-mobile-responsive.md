# Story: Mobile Responsive

## Scope

Ensure the tool works on mobile devices.

## Acceptance Criteria

- [ ] AC1: Landing page readable on 320px width
- [ ] AC2: Assessment flow usable on mobile
- [ ] AC3: Results page fits mobile screen
- [ ] AC4: Buttons touch-friendly (min 44px tap target)
- [ ] AC5: No horizontal scroll on any page

## Technical Notes

- Use Tailwind responsive classes (md:, lg:)
- Test on Chrome DevTools mobile emulation
- Key breakpoints: 320px, 375px, 768px
